# Android Savable Multi-tool Bitmap

[![API](https://img.shields.io/badge/API-14%2B-brightgreen.svg?style=flat)](https://developer.android.com/about/versions/android-4.0)

## Description
A fingerpaint bitmap using [DrawableView](https://github.com/PaNaVTEC/DrawableView) methods for providing a basic multi-tool raster graphics editor.

## Installation
1. Clone the project;
2. unzip the folder;
3. run [Android Studio](https://d.android.com/studio/);
4. open a new project;
5. and pick the package.
